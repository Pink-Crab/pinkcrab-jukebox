# PinkCrab Jukebox - WordPress Block

A full-featured audio jukebox block for WordPress Gutenberg that provides a responsive, customizable music player with advanced playback controls.

## Overview

PinkCrab Jukebox is a WordPress block plugin that transforms your audio content into an interactive, modern music player. It allows you to create playlists with multiple tracks, each with artwork, metadata, and custom styling options.

## Key Features

- **Complete Playback Controls**: Play/pause, previous/next, shuffle, and repeat modes (off/all/one)
- **Queue Management**: Add tracks to a queue, reorder, and manage playback order
- **Track Filtering**: Search and filter tracks by title, artist, or album
- **Visual Customization**: Fully customizable color scheme and artwork display
- **Audio Visualizer**: Multiple visualization modes (bars, oscilloscope, mirror, fire)
- **Responsive Design**: Works seamlessly across all device sizes
- **Keyboard Shortcuts**: Full keyboard navigation support
- **Volume Control**: Adjustable volume with mute functionality
- **Progress Tracking**: Visual progress bar with seek functionality

## Technical Details

- Built for WordPress 6.0+ and PHP 7.4+
- Uses WordPress Block API (API Version 3)
- Modern JavaScript with React components
- HTML5 audio fallback for maximum compatibility
- Fully accessible with ARIA labels and keyboard navigation
- Secure output with proper WordPress sanitization and escaping
